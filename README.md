## ✨ Features
- Animated gradient branding  
- Real-time messaging UI  
- Media sharing support  
- Responsive and modern glass-style design  
- Session status (signed-in user display)  

---

## 🛠️ Tech Stack
- **Frontend:** React, CSS (custom), Firebase   
- **Deployment:** Vercel
